# USB Type-C 接口原理图库与PCB封装库

## 简介

本仓库提供了一个包含多种USB Type-C接口封装的资源文件：`AD type-c接口原理图库、PCB封装库.rar`。该文件内含市面上常见的USB Type-C接口封装，包括侧立式、贴片式、直插式、两脚固定、四角固定等多种类型。资源文件中包含了原理图库（schlib）和PCB封装库（pcblib），方便用户在设计过程中直接使用。
